# 💳 Credit Card Fraud Detection

This project builds and deploys a **credit card fraud detection model** with a **Streamlit web app** so that anyone can try the model interactively.

# Features

- **Machine-learning model** trained on the [Kaggle Credit Card Fraud dataset](https://www.kaggle.com/mlg-ulb/creditcardfraud).
- Handles **highly imbalanced data** (fraud ~0.17% of transactions).
- Exports a ready-to-use `fraud_model.pkl` for inference.
- **Streamlit app** with:
  - Built-in demo dataset
  - CSV batch scoring
  - Single-transaction scoring
  - Adjustable decision threshold and live metrics.

# Project Structure

```
ccFraudDetect/

├── requirements.txt # Python dependencies for training & app
├── .streamlit/
│ └── secrets.toml # secrets for deployment (model/data URLs)

```

# Installation

- Method 1 (preferred):

```bash
    git clone https://github.com/SooYungTing/ccFraudDetect.git
    cd ccFraudDetect
    conda create --name fraud
    conda activate fraud
    conda install -n fraud python=3.13
    python --version
    pip install -r requirements.txt
```

- Method 2 (might have wrong versions):

```bash
    git clone https://github.com/SooYungTing/ccFraudDetect.git
    cd ccFraudDetect
    conda create --name fraud
    conda activate fraud
    conda install -n fraud python=3.13
    python --version
    conda install -c conda-forge py-xgboost
    pip install jupyter ipykernel kagglehub matplotlib plotly pandas seaborn catboost lightgbm streamlit
    pip install scikit-learn==1.5.2
```

# Running the code / training the model

1. Open fraud_detection.ipynb in Jupyter or VS Code and run all cells.
   - This will:
     - Train the model
       Evaluate it (AUROC, AUPRC)
       Export fraud_model.pkl
2. Running the streamlit app (local):
   - Create a `.streamlit/secrets.toml` file
     - Within the `secrets.toml` file, add this:
     ```toml
     [paths]
     model_path = "link/url/to/where/the/model/is/uploaded"
     sample_csv = "link/url/to/where/the/model/is/uploaded/for/here/i/uploaded/it/on/google/drive"
     ```
     - This is an example:
     ```toml
     [paths]
     model_path = "https://raw.githubusercontent.com/SooYungTing/ccFraudDetect/main/fraud_model.pkl"
     sample_csv = "https://drive.google.com/uc?export=download&id=1kn_6rzXoLQyO_T3qOaQy033vByLs6fhp"
     ```
   - `streamlit run streamlit_app.py`
   - Inside the app you can:
     - ▶ Built-in sample data – instantly see predictions using the small demo CSV.
       -📄 Upload CSV – upload your own transactions CSV. If it includes a Class column (0 = normal, 1 = fraud), the app will display evaluation metrics. - Please refer to the kagglehub dataset to find out what each parameter represents.
     - 🧪 Single transaction – manually type feature values and get the fraud probability.
